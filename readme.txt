== Description ==	
Enable to disable srcset :) 
It is necessary to disable default responsive images sometimes, especially in older themes. This plugin will make it easy.  


== Installation ==

- upload 'better-masonry' folder into '...wp-content\plugins\' folder
- go to Dashboard > Plugins and activate 'Better Masonry' plugin, 
- That's all
